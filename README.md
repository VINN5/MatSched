# MatSched

Online matatu booking system in Kenya: Reserve seats, view schedules, and pay via M-Pesa.

## Features
- Search and view matatu routes and schedules
- Interactive seat selection and booking
- Secure payment integration with M-Pesa
- User accounts for passengers drivers and admins
- Real-time availability updates

## Tech Stack
- **Frontend**: React (in `/frontend`)
- **Backend**: Node.js + Express (in `/backend`)
- **Database**: MongoDB 
- **Payments**: M-Pesa API
